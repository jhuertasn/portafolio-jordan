# 🚀 Portafolio Profesional - Jordan Angel Huertas Negrón

Bienvenido al repositorio de mi portafolio personal. Soy un **Técnico egresado en Diseño y Desarrollo de Software** radicado en Lima, Perú, con una sólida base en soporte técnico y una mentalidad orientada a la resolución de problemas técnicos complejos.

## 🛠️ Perfil Técnico
Me especializo en el soporte a usuarios finales, gestión de incidencias Nivel 1 y desarrollo de soluciones escalables. Mi experiencia combina la administración de sistemas con la construcción de arquitecturas modernas.

* **Soporte TI**: Instalación y configuración de software, clonación de sistemas operativos (Clonezilla) y mantenimiento preventivo.
* **Sistemas Operativos**: Dominio de Windows 10/11 y macOS.
* **Bases de Datos**: Gestión y resolución de incidencias en MariaDB y MySQL.
* **Desarrollo**: Creación de aplicaciones con Java (Spring Boot), Node.js, React y Astro.

## 💻 Proyectos Destacados

1. **EcoPoints**: Plataforma integral de reciclaje con arquitectura de microservicios orquestados con Docker Compose y RabbitMQ.
2. **EvaluaApp (Tesis)**: Sistema de evaluación por rúbricas desarrollado con Java 17 y Node.js para la gestión académica.
3. **Comparador de Precios**: Extensión de Chrome desarrollada con TypeScript que utiliza Web Scraping para análisis de datos en tiempo real.
4. **Reproductor Inteligente**: Herramienta para análisis de entrevistas construida con React y Custom Hooks.

## ⚙️ Tecnologías Utilizadas en este Portafolio
Este sitio fue construido siguiendo las mejores prácticas de desarrollo web moderno:

* **Framework**: [Astro](https://astro.build/)
* **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
* **Despliegue**: [Vercel](https://vercel.com/)
* **Seguridad**: Configuración de cabeceras de seguridad mediante `vercel.json` para protección contra clickjacking y control de políticas de referer.

## 🧞 Comandos del Proyecto

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala las dependencias necesarias. |
| `npm run dev` | Inicia el servidor de desarrollo local en `localhost:4321`. |
| `npm run build` | Compila el sitio para producción en la carpeta `./dist/`. |
| `npm run preview` | Previsualiza la compilación localmente antes del despliegue. |
